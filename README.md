# go-gin-teddy-bears-api-v2

## Setup

### Packages
- `swaggo/swag` tool for swagger instigation
    ```bash
    go install github.com/swaggo/swag/cmd/swag@latest
    ```

### Database Setup
```bash
make db
```

## Running the Application

### gin dev
```bash
make gin_dev
```

### fiber dev
```bash
make fiber_dev
```
